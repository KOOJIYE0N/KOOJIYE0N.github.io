# KOO_github_io

Personal GitHub Pages site for Koo Jiyeon / 구지연.

## Deploy

1. Push this folder to `KOOJIYE0N/KOO_github_io`.
2. In GitHub, open `Settings > Pages`.
3. Set the source to the root of the default branch.
4. The site will be available at `https://koojiye0n.github.io/KOO_github_io/`.

To use the root address `https://koojiye0n.github.io/`, rename the repository to
`KOOJIYE0N.github.io`.

## Edit Content

- Main content: `index.html`
- Styling: `styles.css`
- Filtering behavior: `script.js`
- Hero image: `assets/hero-robotics-ai.png`

The publication descriptions are drafted from public listings and should be refined with exact author order, roles, project links, and images when available.
